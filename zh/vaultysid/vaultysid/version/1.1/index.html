VaultysID 1.1

2024年2月29日 · 阅读时间一分钟
Jérôme Delaire
Lead Dev

Toute l'équipe de Vaultys est heureuse de vous présenter un nouvelle version de VaultysID, complètement réécrite pour intégrer notre nouvelle Charte Graphique. Nous avons intégré aussi la possibilité de personnaliser votre avatar et aussi ajouter la sauvegarde de votre compte.

Nouvelle charte graphique​

Le charte graphique de Vaultys a été retravaillée. Couleur, sombre, clair. L'interface de VaultysID change pour respecter cette nouvelle charte !

Avatar​

Vous pouvez désormais utiliser un avatar et le personnaliser pour représenter vos identités Vaultys ! Nous avons choisi quelques chose de simple pour vous représenter. Dans le futur nous allons itérer pour donner encore plus de possibilités.

Importer/Exporter​

Il est désormais possible d'effectuer un backup de vos profils et de les réimporter. Très utile en cas de perte, vol ou casse de votre téléphone, nous recommandons de régulièrement sauvegarder un backup de vos identités Vaultys !
