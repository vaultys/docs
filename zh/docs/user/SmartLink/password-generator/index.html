🔐 密码生成器

您可以从SmartLink的任何页面打开密码生成器。

1. 点击菜单​

2. 点击密码生成器​

3. 复制密码​

您可以点击密码以将其复制到剪贴板中。

4. 生成新密码​

点击“重新生成”按钮以创建新密码。

5. 密码类型​

您可以更改要生成的密码类型。您可以选择传统密码（字符序列）或密码短语（单词序列）。

密码短语​

6. 短语长度​

使用蓝色滑块调整要生成的密码短语中的单词数。

7. 短语语言​

点击语言以选择密码短语中的单词语言。

密码​

8. 密码长度​

使用蓝色滑块选择密码中包含的字符数。

9. 字符类型​

使用复选框选择要包含在密码中的字符类型。

💡 为了最佳安全性​

为了获得最佳安全性，Vaultys建议您至少使用10个字符（最好20个）作为您的密码，并包含所有可能的字符类型（小写字母、大写字母、数字和特殊字符）。

对于密码短语，Vaultys建议使用12个单词（最好24个）。
