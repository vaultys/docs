🔐 密码管理器

1. 点击密码选项卡​

密码选项卡允许您访问密码管理器。

您可以在其中找到您可以为其设置或访问密码的所有应用程序。

2. 用户密码​

通过选择在我们的示例中配置为用户级密码模式的Trello应用程序，我可以设置用户名和密码。

在这种情况下，我将是唯一能够访问此密码的人。

要设置密码，请填写“用户名”和“密码”字段，然后单击“保存”。

3. 复制密码​

现在密码已设置，您随时可以检索它。

要做到这一点，请单击“复制用户名”或“复制密码”按钮。

4. 组织密码​

通过选择在我们的示例中配置为组织唯一密码模式的AWS应用程序，我们注意到我可以访问该应用程序的组织密码，但我无法自行定义密码。

5. 组密码​

通过选择在我们的示例中配置为组密码模式的Asana应用程序，我们注意到我可以访问为我所属组定义的密码。
