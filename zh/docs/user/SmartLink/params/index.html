⚙️ 语言和主题

您可以从您的个人资料访问设置，以更改语言和主题。

1. 点击菜单​

点击屏幕右上角的菜单。

2. 点击个人资料​

3. 更改主题​

您可以根据自己的喜好更改应用程序的主题。

您可以选择明亮模式和暗黑模式。

如果选择系统模式，则根据您系统的设置（Windows/MacOS/Linux/Android/iOS等）应用明亮或暗黑主题。

4. 更改语言​

您还可以更改应用程序的语言（英语，法语，德语，中文）。
