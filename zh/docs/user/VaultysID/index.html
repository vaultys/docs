VaultysID

VaultysID是一种个人多锁钥匙，包含在一个移动应用程序中，任何人都可以安装在自己的手机上。 它可以与任何已设置为接受该个人钥匙的“锁”软件一起使用。例如，您企业的SmartLink管理员将声明接受您的VaultysID作为进入您企业的SmartLink空间的钥匙。 本节适用于希望下载VaultysID并在兼容的产品上安全识别自己的任何人。 它介绍了如何安装VaultysID，可以更改的设置，如何使用VID，如何在手机丢失、出售或被盗时备份以便找回。

安装VaultysID​

  • 数据安全：提供多种选项来保护您的身份，包括PassKey、硬件密钥或直接存储在应用程序中的标识符。每种选项都提供不同级别的安全性和便利性。
  • 安装：VaultysID可安装在iOS、Android或作为渐进式Web应用程序（PWA）。
  • 初始设置：选择语言、选择模式（明亮或暗黑）以及授权访问相机以扫描身份验证所需的QR码。
  • 创建去中心化身份（SSI）：按照引导步骤创建您的SSI，安全选项包括PassKey、FIDO2或根据您特定需求的软件。
