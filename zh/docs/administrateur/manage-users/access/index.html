访问管理

访问用户访问管理页面​

1. 访问管理控制台​

单击SmartLink右上角菜单，然后单击管理员模式以访问管理控制台。

2. 点击用户选项卡​

单击Admin -> 用户选项卡以访问用户管理页面。

3. 点击操作按钮​

查找要管理权限的用户，然后单击相应的操作按钮。

4. 点击访问​

文件夹访问管理​

5. 添加文件夹​

在用户访问管理页面，显示SmartLink的所有文件夹列表，根据用户的权限以不同颜色显示。

灰色文件夹 => 未授权

黄色文件夹 => 已授权

蓝色文件夹 => 已授权，具有管理权限

要授权用户访问文件夹，请将鼠标悬停在文件夹上，然后单击行末的 +。

6. 移除文件夹​

要移除对文件夹的访问权限，请将鼠标悬停在文件夹上，然后单击行末的红色叉号。

文件夹管理权限​

7. 点击文件夹​

要查看有关文件夹的更多配置，请单击该文件夹。

8. 设置管理权限​

侧边栏将显示有关文件夹的更多信息。

单击管理员按钮将用户设置为该文件夹的管理员。

9. 调整权限​

然后，您可以调整用户的权限。

直接共享的应用程序​

10. 点击应用程序​

要管理直接与用户共享的应用程序（而不是通过文件夹），请单击应用程序选项卡。

11. 修改对应用程序的访问权限​

左侧显示与用户共享的应用程序。 右侧显示未共享的应用程序。

要移除/添加对应用程序的访问权限，请单击该应用程序。

复制另一个用户的权限​

12. 点击重置所有​

在复制另一个用户的权限之前，需要重置所有权限。 为此，请单击“重置所有”按钮，然后在打开的窗口中单击“删除”。

13. 点击比较访问权限​

14. 选择另一个用户​

打开一个窗口，显示其他用户的列表。单击其中一个以选择。

15. 复制权限​

屏幕分为两部分。右侧是当前用户的权限，左侧是您刚选择的用户的权限。

单击“复制权限”以将右侧用户的权限复制到左侧用户。

确认删除用户。
