启用/禁用用户

在用户暂时离开的情况下（例如员工停工），您可以禁用他们的 SmartLink 帐户。用户在尝试登录时将被拒绝。

1. 点击用户选项卡​

通过单击管理选项卡 -> 用户，访问用户管理页面。

2. 禁用/启用用户​

然后在列表中查找用户，然后单击“启用”开关以禁用或启用用户帐户。
