一个用户丢失/损坏了手机

在手机丢失或损坏的情况下，您可以将新的 Vaultys 身份关联到您员工/用户的帐户上。

1. 点击“用户”选项卡​

通过单击“管理” -> “用户”选项卡，访问用户管理页面。

2. 访问用户个人资料​

在列表中搜索用户，然后单击用户以访问其个人资料。

3. 点击“关联的 VaultysID”​

在用户个人资料页面，选择“关联的 VaultysID”选项卡。

4. 点击“添加”​

用户的不同身份将显示出来。单击“添加”按钮以启动新身份的配对过程。

5. 配对​

相关用户可以扫描屏幕上显示的 QR 码，以配对他们的新设备。

6. 设置为默认​

新身份将显示在列表中。单击黄色星星将其设置为默认身份。

7. 删除丢失的身份​

出于安全考虑，单击红色垃圾桶删除丢失的身份。

重命名身份（可选）​

8. 点击“编辑”​

您可以重命名新身份以便更轻松管理。请单击蓝色铅笔。

9. 重命名​

输入一个身份名称，例如智能手机型号，以便轻松了解这个身份对应的设备。

单击“保存”以保存新名称。
