使用SmartLink开始

信息

💡 第一次使用Smartlink以获取所有有用信息

准备您的计算机

要开始使用SmartLink，您需要在计算机上安装一个最新版本的浏览器。SmartLink与Google Chrome、Safari、Firefox兼容。其他类型的浏览器也应支持SmartLink，即使没有进行所有测试。

除了浏览器，您只需要互联网连接即可运行SmartLink。SmartLink应用程序对带宽要求不高。

在线注册

要注册，请直接访问网站https://go.vaultys.com，您可以选择演示选项或直接创建免费组织。

演示网站​

如果您直接访问链接https://start.vaultys.com/demo，请直接填写在线表单。在可以登录到演示环境之前，需要验证您的电子邮件。然后您可以探索应用程序，甚至可以直接从“我的SmartLink”菜单中创建自己的组织。

免费创建自己的组织​

如果您直接访问链接https://go.vaultys.com，请直接填写在线表单。在可以进行最终注册之前，需要验证您的电子邮件。然后将直接为您提供个性化的网址。这是您访问新SmartLink的入口。

首次登录

注册后，您将看到一个特殊页面，其中将说明如何无需密码登录。

然后需要在您的智能手机上下载VaultysID应用程序。这个独立于SmartLink的应用程序将帮助您创建您的凭据。有关更多信息，请访问页面安装VaultysID

在SmartLink中导航​

登录后，您将进入您组织的仪表板。在这里，您可以访问所有信息，管理您的设置，并浏览应用程序的不同部分。请花些时间熟悉界面，以便更轻松地使用SmartLink。

访问用户和管理员菜单

作为SmartLink的所有者，您可以访问两种类型的菜单：用户菜单和管理员菜单。

用户菜单​

用户菜单为您提供对SmartLink基本功能的完整访问权限。在此菜单中，您可以访问您的应用程序，管理您的密码，并根据需要创建新的SmartLink。

管理员菜单​

如果您是管理员，还可以访问管理员菜单。此菜单允许您管理SmartLink组织的所有方面。您可以添加或删除用户、应用程序或组，并管理组织的设置等等。

个人资料菜单​

位于界面右上角的个人资料菜单是您在SmartLink上的个人控制中心。单击此菜单，您可以访问与您的个人资料相关的一系列信息和设置。

首先，您会找到您的个人设置。这些设置允许您自定义在SmartLink上的体验，包括修改个人信息、首选项和显示语言。可用的语言有法语、英语、中文和德语。

然后，您有您的登录标识。在这里，您可以管理与您的SmartLink帐户相关的所有信息。

最后，您有浏览器扩展管理。此部分允许您添加、修改或删除浏览器中的SmartLink扩展。这是一个重要功能，可以让您快速访问您的SmartLinks，而无需每次都打开应用程序。
