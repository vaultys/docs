设置您的组织

1. 访问管理控制台​

单击SmartLink右上角的菜单，然后单击管理员模式以访问管理控制台

2. 点击组织​

单击管理选项卡 -> 组织以访问您的组织设置

3. 用户/应用程序/群组​

您可以找到各种信息，包括您可以创建的用户、应用程序和群组数量

4. 更改许可证​

如果您需要更多功能，可以通过单击“升级”按钮订阅另一个许可证。然后，您将被重定向到升级页面

5. 修改语言​

您可以修改组织的默认语言。

组织的语言将成为所有用户的默认语言。

随后，每个用户都可以在其个人设置中选择其他语言。

6. 会话持续时间​

您可以为您的SmartLink设置会话持续时间。 会话持续时间是指用户无需再次扫描QR码而保持登录状态的持续时间。

会话持续时间可在1到24小时之间配置。

7. 登录屏幕类型​

如果需要，您可以选择更简约的登录屏幕，勾选此框。
