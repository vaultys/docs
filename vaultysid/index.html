VaultysID 2.0

14 août 2024 · 2 minutes de lecture
Jérôme Delaire
Lead Dev

Rappels pour effectuer des backups​

Nous vous recommandons de régulièrement effectuer un backup de vos profils VaultysID.

Vos backups vous permettent de restaurer vos identités sur un nouvel appareil en cas de perte, casse, ou vol.

La version 2.0 de VaultysID intègre des rappels pour vous aider à effectuer vos backups régulièrement.

Un rappel sera effectué chaque mois, sauf si vous cochez la case 'Ne plus me demander'.

Plus de paramètres​

Dans les paramètres apparaît désormais une nouvelle langue représentée par un globe. VaultysID empêchera par défaut les traducteurs automatiques (ex google trad), en sélectionnant le globe vous autorisez votre traducteur à traduire l'application. Cela permet donc d'avoir une traduction approximative dans d'autres langues.

Un nouveau menu "Aide" apparaît également. Cliquez dessus pour accéder à la page d'aide de VaultysID :

Simplification de l'onboarding​

Au premier lancement de l'application, la procédure d'enrollement a été grandement simplifiée.

Sécurité​

Le procotole de Vaultys a été mis à jour dans le but d'améliorer toujours plus votre sécurité.

VaultysID 1.2

8 avril 2024 · Une minute de lecture
Jérôme Delaire
Lead Dev

Amélioration de l'interface graphique​

L'interface graphique a été retravaillée pour être plus agréable, plus uniforme et plus animée ! Nous avons notamment utilisé des composants standards basés sur une expérience aboutié de l'expérience utilisateur. Nous sommes certains que vous aprécierez !

Code PIN​

En code PIN est désormais obligatoire pour créer votre identité Vaultys lorsque vous séléctionnez le mode "logiciel" comme type de profil.

VaultysID 1.1

29 février 2024 · Une minute de lecture
Jérôme Delaire
Lead Dev

Toute l'équipe de Vaultys est heureuse de vous présenter un nouvelle version de VaultysID, complètement réécrite pour intégrer notre nouvelle Charte Graphique. Nous avons intégré aussi la possibilité de personnaliser votre avatar et aussi ajouter la sauvegarde de votre compte.

Nouvelle charte graphique​

Le charte graphique de Vaultys a été retravaillée. Couleur, sombre, clair. L'interface de VaultysID change pour respecter cette nouvelle charte !

Avatar​

Vous pouvez désormais utiliser un avatar et le personnaliser pour représenter vos identités Vaultys ! Nous avons choisi quelques chose de simple pour vous représenter. Dans le futur nous allons itérer pour donner encore plus de possibilités.

Importer/Exporter​

Il est désormais possible d'effectuer un backup de vos profils et de les réimporter. Très utile en cas de perte, vol ou casse de votre téléphone, nous recommandons de régulièrement sauvegarder un backup de vos identités Vaultys !
