SmartLink Version 3.4

15 juin 2024 · Une minute de lecture
Hugo Surply
Communication

La principale fonctionalité de cette version est l'arrivée tant attendue de l'administration de groupes. En Effet chaque groupe peut avoir désormais un ou plusieurs administrateurs avec des droits spécifiques (ajout d'applications, d'utilisateurs etc...).

  • Administrateur de groupe
  • Tailles d'icones : sur la page d'acceuil, il est possible de changer la taille des icones en appuyant sur le boutton
  • Erreurs : Une meilleure information est donnée en cas d'erreur de connexion, notamment dans le cas où un utilisateur utilise un téléphone avec une horloge déréglée (ce qui arrive bien plus souvent qu'on ne l'imaginait).

SmartLink Version 3.1

15 février 2024 · Une minute de lecture
Hugo Surply
Communication

La version 3.1 vient avec une série d'améliorations :

  • Panneau de notification : L'administrateur est notifié de demandes par ses collaborateurs d'ajout de nouvelles applications
  • Import d'utilisateurs : L'administrateur peut importer une liste de collaborateurs à partir d'un CSV
  • Indication qu'une app est déjà installéé : Pour éviter les doublons, l'appstore indique désormais à l'administrateur si elle est déjà installée
  • Documentation : Désormais certains menus sont agrémentés d'un petit permettant de renvoyer vers la documentation nécessaire
