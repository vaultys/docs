Extension 1.2.0

11. September 2024 · 2 Minuten Lesezeit
Jérôme Delaire
Lead Dev

La nouvelle version 1.2.0 de l'extension SmartLink embarque une fonctionnalité d'anti-phishing. Cette fonctionnalité permet de bloquer les formulaires de connexion des sites web qui ne sont pas connus de votre SmartLink. Cela évite à l'utilisateur de se faire subtiliser ses identifiants par des sites de phishing.

Les paramètres d'anti-phishing peuvent être définis depuis SmartLink. Il est possible de le désactiver complètement, de choisir une version aggressive et une version plus permissive, ou encore de blacklister complètement certains sites.

Anti Phishing​

Si l'anti-phishing est activé et configuré sur la version plus permissive, cette fenêtre s'affichera par-dessus les formulaires de connexion en provenance des sites qui ne sont pas enregistrés dans SmartLink :

L'utilisateur peut cliquer sur le bouton "Fermer" pour ignorer l'avertissement et se connecter sur le site web en connaissance de cause.

Anti Phishing aggressif​

Dans sa version plus aggressif, l'anti phishing fera apparaître une fenêtre qui ne pourra pas être fermée. Il ne sera alors pas possible de se connecter sur des sites web inconnus :

Blacklist​

Les administrateurs de votre SmartLink peuvent également bloquer complètement l'accès à certains sites, cela bloquera entièrement le site, pas seulement les formulaires de connexion :

Connexion​

Il est maintenant possible de se connecter à l'extension directement depuis n'importe quelle page web ! Désormais si lorsque vous vous rendez sur une page web sans avoir connecté votre extension, et que vous arrivez sur un formulaire de connexion, l'extension SmartLink affichera une fenêtre pour vous y connecter par dessus le formulaire de la page web.

Extension 1.0.8

14. August 2024 · Eine Minute Lesezeit
Jérôme Delaire
Lead Dev

Amélioration de l'auto remplissage​

L'extension SmartLink permet de remplir automatiquement vos noms d'utilisateurs et mots de passe dans vos différentes applications web. Cette fonctionnalité a été entièrement revue et améliorée pour la version 1.0.8.

Nouveau mode de connnexion​

Le mode de connexion a été remplacé par un QR code que vous pouvez directement scanner avec votre application VaultysID. Plus besoin de passer par la version web de votre SmartLink pour accéder à votre extension !

Menu​

Un nouveau menu fait son apparition. Vous pouvez y accéder depuis la barre qui se situe en haut de l'extension, en cliquant sur le nom de votre SmartLink en haut à droite.

Depuis ce menu vous pouvez très facilement switcher entre vos SmartLink si vous en avez plusieurs. Vous également ouvrir votre SmartLink en version web, vous connecter à de nouveaux SmartLink, ou déconnecter votre extension.

Force des mots de passe​

Une indication sur la force des mots de passe que vous choisissez a été ajouté.
