Ein Benutzer hat sein Telefon verloren/zerbrochen

Im Falle eines Verlusts oder einer Beschädigung des Telefons können Sie eine neue Vaultys-Identität mit dem Konto Ihres Mitarbeiters/Benutzers verknüpfen.

1. Klicken Sie auf die Registerkarte Benutzer​

Greifen Sie auf die Benutzerverwaltungsseite zu, indem Sie auf die Registerkarte Admin -> Benutzer klicken.

2. Gehen Sie zum Benutzerprofil​

Suchen Sie den Benutzer in der Liste und klicken Sie auf sein Profil, um darauf zuzugreifen.

3. Klicken Sie auf zugeordnete VaultysID​

Wählen Sie im Benutzerprofil die Registerkarte Zugeordnete VaultysID aus.

4. Klicken Sie auf Hinzufügen​

Die verschiedenen Identitäten des Benutzers werden angezeigt. Klicken Sie auf die Schaltfläche Hinzufügen, um den Vorgang zum Koppeln einer neuen Identität zu starten.

5. Koppeln​

Der betroffene Benutzer kann dann den QR-Code scannen, der auf dem Bildschirm angezeigt wird, um sein neues Gerät zu koppeln.

6. Als Standard festlegen​

Die neue Identität wird dann in der Liste angezeigt. Legen Sie sie als Standard fest, indem Sie auf den gelben Stern klicken.

7. Entfernen Sie die verlorene Identität​

Aus Sicherheitsgründen entfernen Sie die verlorene Identität, indem Sie auf den roten Mülleimer klicken.

Identität umbenennen (optional)​

8. Klicken Sie auf Bearbeiten​

Sie können die neue Identität umbenennen, um die Verwaltung zu erleichtern. Klicken Sie dazu auf den blauen Stift.

9. Umbenennen​

Geben Sie einen Namen für die Identität ein, z. B. das Modell des Smartphones, um leicht zu verstehen, welchem Gerät diese Identität entspricht.

Klicken Sie auf Speichern, um den neuen Namen zu speichern.
