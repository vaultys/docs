SmartLink 3.4

15. Juni 2024 · Eine Minute Lesezeit
Hugo Surply
Kommunikation

Die Hauptfunktion dieser Version ist die lang erwartete Einführung der Gruppenverwaltung. Jede Gruppe kann nun einen oder mehrere Administratoren mit spezifischen Rechten haben (Hinzufügen von Anwendungen, Benutzern usw.).

  • Gruppenadministrator
  • Symbolgrößen: Auf der Startseite können Sie die Symbolgrößen ändern, indem Sie auf die Schaltfläche drücken.
  • Fehler: Es wird eine bessere Information bei Verbindungsfehlern bereitgestellt, insbesondere wenn ein Benutzer ein Telefon mit einer falsch eingestellten Uhr verwendet (was häufiger vorkommt, als man denkt).

SmartLink 3.3

15. Mai 2024 · Eine Minute Lesezeit
Hugo Surply
Kommunikation

Ein großer Teil der Entwicklung besteht darin, den Serverteil erheblich zu verbessern, um die Anwendung zuverlässiger zu machen und die Entwicklungszeit für die Integration neuer Funktionen zu verkürzen.

  • Sicherheit: Noch weitergehende Stärkung der Sicherheitsmaßnahmen und Datenverschlüsselung
  • UI: Verbesserung der Benutzeroberfläche, um die Nutzung aller angebotenen Funktionen zu erleichtern
  • Onboarding: Einführung eines neuen Onboarding-Prozesses, um neuen Benutzern den Einstieg in die Vaultys-Tools zu erleichtern. Sie können es unter https://go.vaultys.com testen.

SmartLink 3.1

15. Februar 2024 · Eine Minute Lesezeit
Hugo Surply
Kommunikation

Die Version 3.1 wird mit einer Reihe von Verbesserungen geliefert:

  • Benachrichtigungspanel: Der Administrator wird über Anfragen seiner Mitarbeiter zur Hinzufügung neuer Anwendungen informiert
  • Benutzerimport: Der Administrator kann eine Liste von Mitarbeitern aus einer CSV-Datei importieren
  • Anzeige, dass eine App bereits installiert ist: Um Duplikate zu vermeiden, zeigt der App Store dem Administrator nun an, ob die App bereits installiert ist
  • Dokumentation: Einige Menüs sind jetzt mit einem kleinen versehen, der auf die erforderliche Dokumentation verweist
