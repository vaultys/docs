SmartLink 3.6

September 19, 2024 · 2 min read
Jérôme Delaire
Lead Dev

Une vue utilisateur et une vue administrateur​

Avec la version 3.6 de SmartLink, la vue utilisateur est désormais totalement dissociée dans la vue administrateur.

Que vous soyez un simple utilisateur ou un administrateur, vous arrivez sur la même vue :

Si vous êtes administrateur, vous pouvez ensuite switcher entre la vue utilisateur et la vue administrateur via le menu en haut à droite :

La couleur de la vue administrateur change, pour ne pas s'y perdre !

Les groupes deviennent des dossiers​

La notion de groupes dans SmartLink a été étendue et remplacé par des dossiers.

Tout commes les groupes, les dossiers vous permettent de gérer facilement les accès dans votre SmartLink. Mais les dossiers vous permettent de créer une structure plus complexe en imbriquant des dossiers dans d'autres dossiers.

Les droits d'un dossier se reflètent sur ses sous-dossiers.

Cas d'usage​

Vous pouvez créer la structure que vous voulez, celle qui représente le mieux le fonctionnement de votre entreprise.

Exemple, mon entreprise est composée de plusieurs agences. Chaque agences s'occupe de plusieurs clients. J'ai besoin de partager des accès différents avec chacuns de mes clients. Je peux alors créer un structure de la forme suivante :

📂AgenceA
 ┣ 📂ClientA
 ┃ ┣ 📂Kevin
 ┃ ┗ 📂Hugo
 ┣ 📂ClientB
 ┗ 📂ClientC
 📂AgenceB

Je peux ensuite affecter un gérant sur le dossier AgenceA et un autre sur le dossier AgenceB. Je peux autoriser des utilisateurs sur toute une agence, ou sur le clientA uniquement, etc.

Une page pour gérer les accès​

Il est maintenant possible de gérer facilement les accés d'un utilisateur via une nouvelle page.

Définissez facilement les droits d'accès (dossiers, applications) d'un utilisateur.

Il est également possible depuis cette page de copier les accès d'un utilisateur à un autre

SmartLink 3.4

June 15, 2024 · One minute read
Hugo Surply
Communication

La principale fonctionalité de cette version est l'arrivée tant attendue de l'administration de groupes. En Effet chaque groupe peut avoir désormais un ou plusieurs administrateurs avec des droits spécifiques (ajout d'applications, d'utilisateurs etc...).

  • Administrateur de groupe
  • Tailles d'icones : sur la page d'acceuil, il est possible de changer la taille des icones en appuyant sur le boutton
  • Erreurs : Une meilleure information est donnée en cas d'erreur de connexion, notamment dans le cas où un utilisateur utilise un téléphone avec une horloge déréglée (ce qui arrive bien plus souvent qu'on ne l'imaginait).

SmartLink 3.1

February 15, 2024 · One minute read
Hugo Surply
Communication

La version 3.1 vient avec une série d'améliorations :

  • Panneau de notification : L'administrateur est notifié de demandes par ses collaborateurs d'ajout de nouvelles applications
  • Import d'utilisateurs : L'administrateur peut importer une liste de collaborateurs à partir d'un CSV
  • Indication qu'une app est déjà installéé : Pour éviter les doublons, l'appstore indique désormais à l'administrateur si elle est déjà installée
  • Documentation : Désormais certains menus sont agrémentés d'un petit permettant de renvoyer vers la documentation nécessaire
