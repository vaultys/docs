Installez VaultysID

Matériel nécessaire​

VaultysID est une application développée par Vaultys. Elle est compatible avec iOS et Android. Nous conseillons d'utiliser une version iOS 16.4 ou ultérieure et Android 12 ou ultérieure. Si vous avez acheté votre téléphone il y a moins de trois ans, il devrait être compatible sans modification nécessaire.

Sécurité renforcée avec la Clé Hardware​

Si vous souhaitez utiliser la fonctionnalité clé hardware, vous aurez besoin d'une clé ou d’une carte de type Fido2. Les principales marques sont Yubikey, Feitian, Thétis et le français Neowave. Assurez-vous de vérifier la compatibilité de cette clé avec votre téléphone.

Sécurité renforcée avec Passkey​

VaultysID est compatible avec le protocole Passkey. Pour utiliser Passkey, vous devez créer un compte auprès d'un fournisseur. En général, votre Passkey est déjà disponible. Si vous avez un compte Google sur Android ou iCloud sur iPhone, aucune configuration supplémentaire n'est nécessaire. Si vous rencontrez des problèmes, consultez la documentation sur le site du fabricant de votre téléphone.

Sécurité de vos données​

Vaultys ne collecte à aucun moment vos données personnelles, ni vos identifiants via cette application. Cependant, il est important de comprendre comment vos données sont traitées en fonction des options proposées :

  • Si vous utilisez PassKey, la sécurité est garantie par votre fournisseur. Par exemple, si votre fournisseur est Google, votre clé cryptographique est chiffrée et sauvegardée sur votre drive. La sécurité est gérée par Google. Cela peut poser des problèmes de souveraineté. Si Google ferme votre compte pour une raison ou une autre, vous perdez votre Passkey, et donc votre identifiant VaultysID. Vérifiez donc les conditions d'utilisation publiées par votre fournisseur de Passkey. C'est toutefois l'option que nous recommandons pour la plupart des utilisateurs, car elle représente le meilleur compromis entre sécurité et simplicité.
  • Si vous utilisez la clé hardware, c'est la meilleure option pour votre sécurité et vie privée. Toutes les données de connexion sont physiquement sous votre contrôle. Cependant, cela peut compliquer l’utilisation car vous devez toujours avoir votre clé à portée de main lorsque vous souhaitez vous connecter. C'est l'option que nous recommandons pour les utilisateurs nécessitant une sécurité avancée, comme les administrateurs ou les décideurs.
  • Si aucune des deux options n'est possible pour vous, nous proposons la possibilité de créer un identifiant VaultysID directement stocké dans l'application. Vous avez le contrôle total de votre identifiant, mais la sécurité n'est pas optimale car quiconque a accès à votre téléphone peut accéder à votre identifiant. Nous recommandons cette solution pour des tests ou pour des accès pour lesquels la sécurité n'est pas primordiale.

Installation

Android​

L'application est disponible pour Android sur le Play Store à cette adresse : https://play.google.com/store/apps/details?id=net.vaultys.wallet.

iOS​

L'application VaultysID sera disponible sur l'App Store dans un futur proche. Nous travaillons activement pour rendre cette application accessible à tous les utilisateurs d'iPhone et d'iPad dès que possible. Restez à l'écoute pour plus d'informations.

PWA​

Rendez-vous avec votre téléphone sur cette adresse:

https://wallet.vaultys.net

Pour les utilisateurs qui ont acquis une certaine maîtrise et qui souhaitent aller plus loin dans la gestion de leurs données, il est possible d'installer VaultysID en tant qu'Application Web Progressive (PWA). Cette option offre une plus grande souveraineté, en vous permettant de contrôler davantage votre environnement numérique. Cependant, il est important de noter qu'il existe quelques restrictions à cette méthode.

Configuration

Lors du premier lancement de l'application, plusieurs configurations initiales seront nécessaires :

  • Choix de la langue : L'application vous demandera de choisir la langue que vous souhaitez utiliser. Nous vous recommandons de choisir la langue dans laquelle vous vous sentez le plus à l'aise pour comprendre les instructions et les informations de l'application.
  • Mode sombre : Vous aurez également la possibilité de choisir entre le mode clair et le mode sombre, en fonction de vos préférences. Le mode sombre peut être plus confortable pour vos yeux, surtout si vous utilisez l'application dans un environnement peu éclairé.
  • Autorisation d'accès à la caméra : Enfin, l'application vous demandera l'autorisation d'accéder à la caméra de votre téléphone. Cette autorisation est nécessaire car l'application utilise la caméra pour scanner les codes QR, qui sont une partie essentielle du processus d'authentification.

Création de votre Identité Numérique Décentralisée (SSI)

Pour créer votre identité numérique décentralisée (SSI) avec VaultysID, vous devez suivre quelques étapes simples. Laissez-vous guider !

Configurer vos informations personnelles : Entrez votre nom, votre adresse e-mail et votre numéro de téléphone. Ces informations sont nécessaires pour configurer votre SSI.

Choisir votre option de sécurité : Vous pouvez choisir entre Passkey, FIDO2 ou Software. Chaque option a ses propres avantages et inconvénients, alors choisissez celle qui convient le mieux à vos besoins :

PassKey : Cette option est recommandée pour la plupart des utilisateurs car elle représente le meilleur compromis entre sécurité et simplicité. Votre clé cryptographique est chiffrée et sauvegardée par votre fournisseur (par exemple, Google sur Android ou iCloud sur iPhone).

FIDO2 : Cette option vous offre le plus haut niveau de sécurité et de confidentialité. Toutes les données de connexion sont physiquement sous votre contrôle, stockées sur une clé hardware FIDO2. Cependant, vous devez toujours avoir votre clé à portée de main pour vous connecter.

Software : Si aucune des deux premières options n'est possible pour vous, vous pouvez stocker votre identifiant VaultysID directement dans l'application. Cette option n'offre pas le même niveau de sécurité que les deux autres, mais elle vous donne le contrôle total de votre identifiant.

Après avoir choisi votre option de sécurité, vous aurez créé avec succès votre Identité Numérique Décentralisée avec VaultysID.

Vous êtes désormais prêt à vous connecter à SmartLink grâce à votre nouvelle identité numérique décentralisée créée avec VaultysID. Nous vous recommandons donc de prendre contact avec l’administrateur du SmartLink afin de recevoir une requête d'authentification par courrier électronique. Une fois que vous avez reçu cette requête, vous pourrez l'utiliser pour vous connecter à SmartLink et commencer à utiliser pleinement votre nouvelle identité numérique.

Avec VaultysID, la connexion est simplifiée à l'extrême. Plus besoin de se souvenir de nombreux mots de passe ou d'aller dans les paramètres de votre compte pour se connecter. Tout ce que vous avez à faire c'est de scanner un QR code. C'est aussi simple que cela ! Il suffit d'ouvrir votre application VaultysID et de scanner le QR code fourni. En un instant, vous êtes connecté. Il s'agit d'une solution rapide, simple et sécurisée qui met fin aux mots de passe complexes et aux processus de connexion compliqués.
