Un utilisateur a perdu/cassé son téléphone

En cas de perte ou de casse de téléphone, vous pouvez associer une nouvelle identité Vaultys au compte de votre employé/utilisateur.

1. Cliquez sur l'onglet Utilisateurs​

Accédez à la page de gestion des utilisateurs en cliquant sur l'onglet Admin -> Utilisateurs.

2. Accédez au profil de l'utilisateur​

Rechercher l’utilisateur dans la liste, et cliquez dessus pour accéder à son profil.

3. Cliquez sur VaultysID associés​

Depuis le profil de l'utilisateur, sélectionnez l'onglet VaultysID associés.

4. Cliquez sur Ajouter​

Les différentes identités de l'utilisateur apparaissent. Cliquez sur le bouton Ajouter pour lancer la procédure d'appairage d'une nouvelle identité.

5. Apparairage​

L'utilisateur concerné peut alors scanner le QR code qui apparaît à l'écran pour appairer son nouvel appareil.

6. Définir par défaut​

La nouvelle identité apparaît alors dans la liste. Définissez la par défaut, en cliquant sur l'étoile jaune.

7. Supprimez l'identité perdue​

Par mesure de sécurité, supprimez l'identité perdue en cliquant sur la poubelle rouge.

Renommer l'identité (optionnel)​

8. Cliquez sur modifier​

Vous pouvez renommer la nouvelle identité pour faciliter la gestion. Pour cela cliquez sur le crayon bleu.

9. Renommez​

Entrez un nom pour l'identité, cela peut être par exemple le modèle du smartphone pour comprendre facilement à quel appareil cette identité correspond.

Cliquez sur Enregistrer pour sauvegarder le nouveau nom.
