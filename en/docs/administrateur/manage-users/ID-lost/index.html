A user has lost/broken their phone

In case of loss or breakage of a phone, you can associate a new Vaultys identity with your employee/user's account.

1. Click on the Users tab‚Äã

Access the user management page by clicking on the Admin -> Users tab.

2. Access the user's profile‚Äã

Search for the user in the list, and click on it to access their profile.

3. Click on Associated VaultysIDs‚Äã

From the user's profile, select the Associated VaultysIDs tab.

4. Click on Add‚Äã

The different identities of the user will appear. Click on the Add button to start the pairing process of a new identity.

5. Pairing‚Äã

The user can then scan the QR code displayed on the screen to pair their new device.

6. Set as default‚Äã

The new identity will then appear in the list. Set it as default by clicking on the yellow star.

7. Delete the lost identity‚Äã

For security reasons, delete the lost identity by clicking on the red trash can.

Rename the identity (optional)‚Äã

8. Click on Edit‚Äã

You can rename the new identity for easier management. To do this, click on the blue pencil.

9. Rename‚Äã

Enter a name for the identity, for example, the model of the smartphone to easily understand which device this identity corresponds to.

Click on Save to save the new name.
