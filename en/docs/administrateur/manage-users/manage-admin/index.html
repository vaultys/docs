Gestion des administrateurs

Pour ajouter un administrateur, il suffit d'inviter un utilisateur par email ou de l'ajouter en direct, puis de modifier ses droits via l'onglet Utilisateurs.

1. Cliquez sur l'onglet Utilisateurs​

Cliquez sur l'onglet Admin -> Utilisateurs pour accéder à la page de gestion des utilisateurs.

2. Modifiez les droits​

Recherchez ensuite un utilisateur dans la liste, et modifier les droits administrateurs en activant/désactivant le switch Administrateur.
