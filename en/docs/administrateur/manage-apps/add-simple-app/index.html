Ajouter une application simple

1. Cliquez sur Ajouter​

Accédez à la page de gestion des applications en cliquant sur l'onglet Admin -> Applications.

2. Cliquez sur Ajouter​

Cliquez sur le bouton Ajouter, à côté de la barre de recherche.

3. Créez l'application​

Entrez le nom de votre application et éventuellement une description.

4. Terminé​

Votre application est ajoutée à SmartLink, vous êtes alors redirigé vers la page de configuration de cette application. Vous pouvez alors ajouter une icône ou encore modifier les droits d'accès à cette application.
