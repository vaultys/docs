Créer son SmartLink

1. Rendez-vous sur go.vaultys.com​

Pour créer votre SmartLink rendez-vous sur https://go.vaultys.com .

Si vous n'avez pas encore d'identité Vaultys, cliquez sur Non, et suivez les étapes 2 à 4.

2. Choisissez votre OS​

Choisissez le système d'exploitation de votre smartphone parmis Android, iOS, ou autre.

3. Installez VaultysID​

Installez ensuite VaultysID sur votre smartphone en suivant les indications. Vous pouvez également scanner le QR code affiché à l'écran pour vous rendre directement sur la page de téléchargement de l'application.

Cliquez ensuite sur "Suivant".

4. Créez votre identité dans VaultysID​

Quelques captures d'écrans vous aideront ensuite à créer votre identité dans VaultysID. Cliquez ensuite sur "Suivant".

5. Scannez le QR code​

Pour lier votre identité sur go.vaultys.com , scannez le QR code avec votre application VaultysID.

6. Mon dashboard​

Cliquez ensuite sur "Aller sur mon dashboard"

7. Vos SmartLinks​

Vous arriver alors sur une nouvelle page sur laquelle vous pouvez éventuellement retrouver les SmartLinks qui correspondent à l'identité Vaultys avec laquelle vous êtes connecté.

Vous pouvez également créer un nouveau SmartLink en cliquant sur "Créer un SmartLink".

8. Créer votre SmartLink​

Il vous suffit alors de remplir le formulaire pour créer votre SmartLink.

S'il s'agit de votre premier SmartLink avec cette identité Vaultys, vous serez amené à entrer un email qui sera vérifié via un code à six chiffres.
