Activer/Désactiver un utilisateur

En cas d'absence temporaire d'un utilisateur (arrêt de travail d'un employé par exemple), vous pouvez désactiver son compte SmartLink. L'utilisateur sera alors refusé lorsqu'il tentera de se connecter.

1. Accédez à la console administration​

Cliquez sur le menu en haut à droite de SmartLink puis sur Mode administrateur pour accéder à la console administration

2. Cliquez sur l'onglet Utilisateurs​

Accédez à la page de gestion des utilisateurs en cliquant sur l'onglet Admin -> Utilisateurs.

3. Désactivez/Activez l'utilisateur​

Recherchez ensuite l'utilisateur dans la liste, puis cliquez sur le switch Actif pour désactiver ou activer le compte de l'utilisateur.
